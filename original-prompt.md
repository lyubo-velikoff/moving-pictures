# Original Prompt

Want to create a website that will advertise popular tv shows e.g. moving pictures.

Let's start with just 1 popular tv show "speed and love" chinese tv show.

I want you to gather all information about the tv show from web, research it, gather information such as:
- actors
- trailer
- tv show information that you can think of
- photos of pretty much everything
- episode breakdown
- ratings from popular tv show rating websites


Requirements:
- no mock data, only real data
- keep data layer maybe in a server that we can deploy both server and ui with vercel free tier
- can use create next app to create the app
- both server and ui should have unit tests
- break down the plan with smaller achievable tasks and testable and validating tasks
- design should be stunning, teatrical, very animating, like full screen things etc, feel free to research designs and so on
- must verify and test the app and also ui test as well that things are working
- consider how we are going to fetch up to date information about the tv show too so that our info does not become out of date
- consider us having a like / watched tv show system too
- create a repo, can use github cli to do that
- code should be well organised, easy to maintain, read by human / llm and keep best practices
